20. Valid Parentheses
💚 Easy
Topics: String Stack

解題思路

s = "({}])"

i = 0，stack = ['(']
i = 1，stack = ['(', '{']
i = 2 碰到 } 的時候，從 stack 檢查：

從 stack 的尾巴找到 } -> ✅ ，移除尾巴元素 }，但表有成對

i = 3 碰到 ] 的時候，從 stack 檢查：

從 stack = ['('] 的尾巴找不到] -> ❌

解題步驟

遍歷陣列：

  • 當遇到左括號時，將符號 append 到 Stack
  • 當遇到右括號時，stack 尾巴是否存在對應的左括號
    • 存在：將左括號 pop 出來
    • 不存在：返回 False
  • 檢查 stack 若為空陣列，返回 True ; 反之為 False

複雜度

  • 時間複雜度: O(N)
  • 空間複雜度: O(N)
